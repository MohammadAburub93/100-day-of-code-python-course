## üíª How to Run These Scripts
Since each day's project is isolated, dependencies may vary.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/MohammadAburub93/100-day-of-code-python-course.git
    ```
2.  **Navigate to a specific project:**
    ```bash
    cd Day_25_US_States_Game
    ```
3.  **Install dependencies (if a requirements file exists):**
    ```bash
    pip install -r requirements.txt
    ```
4.  **Run the application:**
    ```bash
    python main.py
    ```
